# Surf Map Keyboard Navigation Guide

This document provides a comprehensive guide to the keyboard navigation features implemented in the interactive surf map. These features are designed to provide full accessibility and a richer user experience for keyboard-only users.

## General Navigation

| Key | Action | Description |
| :--- | :--- | :--- |
| **Arrow Keys** | Pan the map | Use the `↑`, `↓`, `←`, and `→` keys to pan the map in any direction. Holding down a key will result in a smooth, continuous pan. |
| **`+` / `-`** | Zoom | Use the `+` key to zoom in and the `-` key to zoom out. |
| **`0`** | Reset View | Resets the map to its initial default position and zoom level. |
| **`?`** | Toggle Shortcuts | Toggles the visibility of the keyboard shortcuts help panel. |

## Panel and Modal Interaction

| Key | Action | Description |
| :--- | :--- | :--- |
| **`Escape`** | Close Panel | Closes any currently open panel or modal, including the surf spot details panel and the keyboard shortcuts panel. |
| **`Ctrl` + `/`** | Toggle Search Panel | Toggles the visibility of the collapsible search and filter panel on the left. |

## Accessibility Features

*   **Focus Management**: The system is designed with logical focus trapping, ensuring that keyboard focus remains within the map or active panels and does not get lost.
*   **ARIA Attributes**: Proper ARIA (Accessible Rich Internet Applications) attributes have been used throughout the surf map components to ensure compatibility with screen readers.
*   **Screen Reader Announcements**: Key actions, such as opening or closing panels, are announced by screen readers to provide context to visually impaired users.
*   **High Contrast and Reduced Motion**: The UI respects users' system-level preferences for high contrast and reduced motion.

This keyboard navigation system ensures that all core functionalities of the surf map are accessible without requiring a mouse, providing an inclusive experience for all users.
