- **Instituto Hidrográfico de la Marina**: Spanish marine charts

## Source Verification Checklist

### Reliability Indicators
✓ Official government or maritime authority
✓ Professional surf organization with editorial standards
✓ Recent updates with verification dates
✓ Multiple independent sources confirming same coordinates
✓ Consistent with satellite imagery and maps

### Red Flags
⚠ User-generated content without verification
⚠ Outdated information (pre-2010)
⚠ Conflicting coordinates from same source
⚠ Vague location descriptions
⚠ No attribution or source citations

## Cross-Reference Verification

### Primary Cross-Check Sources
1. **Google Maps**: Satellite imagery and street view validation
2. **OpenStreetMap**: Community-validated geographic data
3. **Google Earth**: High-resolution satellite imagery
4. **Marine Charts**: Official nautical navigation data

### Geographical Consistency Checks
- Coastline alignment with described features
- Proximity to claimed towns/landmarks
- Bathymetry (depth) data for reef breaks
- Coastal orientation for wave direction validation

## Source Quality Scoring

### Scoring System (1-10)
- **Official Government Data**: 9-10
- **Marine Charts**: 9-10
- **Professional Surf Platforms**: 7-8
- **Local Tourism Sites**: 6-7
- **Surf Schools/Guides**: 5-6
- **Community Sources**: 3-4
- **Social Media**: 1-3

### Confidence Thresholds
- **Verified**: Average score ≥8 with ≥3 sources
- **High**: Average score ≥7 with ≥2 sources
- **Medium**: Average score ≥5 with ≥1 source
- **Low**: Average score <5 or single unreliable source
