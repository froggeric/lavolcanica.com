# GPS Coordinate Researcher

A comprehensive skill for researching and verifying GPS coordinates with high confidence through multi-language web search, source verification, and map validation.

## Usage

Use this skill when you need to find or verify GPS coordinates for locations with high accuracy requirements. Particularly useful for surf spots, landmarks, or geographical features where precision is critical.

## Core Methodology

### 1. Context-Aware Research
- Extract all available contextual information (names, descriptions, features)
- Use alternative names, descriptions, and practicalities for comprehensive search
- Leverage geographical and environmental characteristics

### 2. Multi-Language Web Search
- Execute searches in English, Spanish, German, Italian, French
- Use language-specific search patterns and terminology
- Target specialized sources based on context

### 3. Source Verification & Cross-Referencing
- Validate coordinates across minimum 3 independent sources
- Check consistency between specialized and general sources
- Verify against official mapping and government data

### 4. Map Validation
- Cross-reference with Google Maps, OpenStreetMap, satellite imagery
- Ensure coordinates align with location descriptions
- Validate geographical plausibility

### 5. Confidence Assessment
- **Verified**: 3+ independent sources + map validation confirmed
- **High**: 2+ reliable sources + map validation confirmed
- **Medium**: Single reliable source or conflicting but resolvable sources
- **Low**: Limited sources, requires manual review

## Reliable Source Types

1. **Specialized Surf Guides**: Surfline, MagicSeaweed, local surf schools
2. **Official Sources**: Government coastal data, marine charts
3. **Local Tourism**: Official tourism websites, local guides
4. **Outdoor Recreation**: Hiking/mapping sites, satellite imagery forums
5. **Academic/Research**: Geological surveys, coastal studies

## Implementation Notes

- Always validate coordinates against geographical descriptions
- Cross-reference spot characteristics with coastal features
- Verify proximity to claimed landmarks/towns
- Document all sources and reasoning for confidence assessment
- Flag locations with conflicting information for manual review
