Surf Map Fixes Verification

Fixes Applied

Issue 1: Search results not opening surf spot panel FIXED

Added showSurfSpotPanel call to search result click handler

Issue 2: Surf spot icons too small and positioned incorrectly FIXED

Increased marker size from 20px to 30px and hover radius from 15px to 25px

Issue 3: Search results click not opening panel FIXED

Same fix as Issue 1 - both were caused by missing panel opening call

Root Cause Analysis

Issue 1 & 3 Root Cause: The search result click callback in surf-map.html was only calling `surfMap.focusOnSpot(spot.id)` but not `window.showSurfSpotPanel(spot.id)`.
Issue 2 Root Cause: The marker size (20px) and hover radius (15px) were too small for comfortable interaction, making markers appear tiny and difficult to click.
